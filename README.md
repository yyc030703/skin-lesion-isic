# Skin Lesion Analysis — ISIC Dataset

Two-problem deep learning pipeline:
1. Classification (ResNet-50 vs EfficientNet-B4)
2. Segmentation (U-Net vs DeepLabV3+)

## Team
- Ellie Yang — Classification
- Szuyu Chi — Segmentation
- Lucia Huang — Infra, viz analysis, integration

## Setup
1. Clone this repo
2. Install requirements: `pip install -r requirements.txt`
3. Dataset lives in Google Drive (link in team Slack)

## Structure
- `src/` — Python modules
- `notebooks/` — EDA and experiments
- `configs/` — Training configs
- `reports/` — Report drafts and figures
